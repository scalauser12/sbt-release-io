# Documentation

[sbt-release-io](https://github.com/scalauser12/sbt-release-io) is a cats-effect IO port of [sbt-release](https://github.com/sbt/sbt-release) for sbt, with composable error handling and resource safety.

## Guides

| Guide | Description |
| ----- | ----------- |
| [**Core (single-project)**](core/README.md) | `sbt-release-io` — release steps, version files, custom steps and plugins |
| [**Monorepo**](monorepo/README.md) | `sbt-release-io-monorepo` — per-project versioning, change detection, tagging |
